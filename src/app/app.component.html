Jenkins Deployment Steps & Considerations -- Staging

1. Setup Jenkins

  • Install Jenkins (Linux/Windows/Container)
  • Install necessary plugins (Git, NodeJS, Netlify, etc.)
  • Configure credentials (GitHub, Netlify tokens)

2. Create Jenkins Pipeline

  • Use Declarative or Scripted Pipeline
  • Add parameters: BRANCH_NAME, ENV, DEPLOY_VERSION
  • Stages: Checkout → Build → Deploy

3. Build & Deploy Logic

  • Install Node.js, run npm install
  • Build Angular using ng build --configuration=ENV
  • Use Netlify CLI or EC2 scripts to deploy

4. Rollback Strategy

  • Backup previous deployment version
  • Use Netlify CLI or EC2 snapshots to rollback
  • Log rollback info in a file or console

5. Notifications & Logs

  • Send email on success/failure
  • Configure Jenkins Email plugin
  • Maintain build logs & artifacts
